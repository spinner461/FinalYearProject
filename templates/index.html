About This Project

This project was developed in Dublin City University, as part of a Final Year Project module in the B.Sc of Applied Physics programme. The project is based on stochastic financial modelling using Brownian Motion as the random element of the stock's growth.

Stochastic Forecasting

Milstein Approximation

The Milstein Numerical approximation is an improvement upon the Euler. It includes second-order terms which tend to make the approximation of random stochastic processes more accurate.
